Descargar Visor Web

El Visor Web es una herramienta esencial para la impresión eficiente de tickets y como contenedor de aplicaciones web. Descargue ahora para mejorar su flujo de trabajo.

  1. Descargue el instalador haciendo clic en el botón de abajo.
  2. Ejecute el archivo descargado.
  3. Siga las instrucciones de instalación en pantalla.
  4. Si te aparece que es un archivo sospechoso, has click sobre el y después a descargar archivo sospechoso. Esto aparece porque no cuento una una firma digital que muchos navegadores piden, pero es totalmente seguro
